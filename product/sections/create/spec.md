# Create Section Specification

## Overview
The heart of Porizō — a wizard-driven flow that guides users through creating personalized songs and poems. Users select a creation type, choose an occasion, capture their story through guided questions, review AI-generated content, and receive a polished final product.

## User Flows

### Primary Flow: Create a Personalized Song
1. User taps Create button (center tab bar button)
2. Type selection screen: Choose "Personalized Song" or "Custom Poem"
3. Story Wizard (7 steps):
   - **Who**: Enter recipient's name (required)
   - **Occasion**: Select from grid of 10 occasions
   - **Memory**: Describe the ONE specific memory to capture (min 10 chars)
   - **AI Questions**: Answer 3 contextual follow-up questions generated by AI
   - **Nicknames**: Optional — add special phrases or inside jokes
   - **What Makes Special**: Optional — describe what makes them special
   - **Review**: Confirm details and select music style
4. Creating Track: Loading animation while track/version is created
5. Lyrics Review: View AI-generated lyrics with section-by-section editing
6. Render & Play: Preview render with progress, then playback controls
7. Completion: Options to share, save, or create another

### Primary Flow: Create a Custom Poem
1. User taps Create button
2. Select "Custom Poem"
3. Story Wizard (similar to song, adjusted prompts):
   - **Who**: Enter recipient's name
   - **Occasion**: Select occasion
   - **Message**: What do you want to express?
   - **AI Questions**: Contextual follow-ups about the relationship
   - **Tone**: Select poem tone (heartfelt, playful, formal, etc.)
   - **Review**: Confirm details
4. Poem Generation: Loading while AI creates poem
5. Poem Review: View generated poem with line-by-line editing
6. Completion: Styled poem card ready to share

### Secondary Flow: Resume Draft
1. User taps draft from Songs/Poems tab
2. Flow resumes at appropriate step (Lyrics/Poem Review)
3. Continue through to completion

### Secondary Flow: Start from Explore
1. User taps occasion card in Explore tab
2. Create flow opens with occasion pre-selected
3. Skips occasion step in wizard

### Secondary Flow: Create Variation
1. User taps "Create Variation" on existing song/poem
2. Create flow opens with context pre-filled
3. Option to modify style, tone, or regenerate content

## UI Requirements

### Modal Presentation
- Full-screen modal (covers tab bar completely)
- Cancel button only on type selection screen
- Back navigation through all wizard steps
- No way to accidentally dismiss mid-creation

### Type Selection Screen
- "What would you like to create?" header
- Two large cards:
  - Song card with music note icon, title, description
  - Poem card with book icon, title, description
- Clear visual hierarchy

### Story Wizard
- Progress bar at top (percentage complete)
- Step title in navigation
- Large, readable prompt text
- Input fields with placeholder examples
- Next/Skip/Back buttons in consistent footer position
- Skip only available for optional steps

### Loading States
- Circular progress with percentage for renders
- Animated icon (waveform for songs, quill for poems)
- Contextual status messages
- Estimated time remaining when available

### Review Screens
- Section-by-section display for lyrics
- Edit button per section
- Inline editing via sheet
- Add/remove lines capability
- "Try Different" regeneration option
- Clear approve/continue CTA

### Player/Completion
- Large play/pause button
- Progress bar with time stamps
- Share button
- "Create Another" option
- "Done" to return to library

## Music Styles (Songs)
Pop, Acoustic, Soul, Folk, Jazz, R&B, Rock, Country, Afrobeats, Highlife, Afropop, Reggaeton, Salsa, Bossa Nova, Bachata, Latin Pop

## Poem Tones (Poems)
Heartfelt, Playful, Formal, Poetic, Simple, Rhyming, Free Verse

## Occasions (Both)
Birthday, Anniversary, Thank You, I Love You, Wedding, Graduation, Celebration, Apology, Encouragement, Custom
