## How to read the data

**In Plausible / GA4:**
1. Filter by `utm_campaign=mothers_day_2026` to see total campaign traffic
2. Group by `utm_source` to compare TikTok vs Pinterest vs Reddit vs ASA
3. Group by `utm_content` to compare creative variants within a channel
4. Set `/download` clicks as the conversion event — that's the install proxy

**Decision rules from the tracking checklist:**
- Channel CPI > $15 sustained 7 days → cut
- Creative variant engagement < 1% for 14 days → kill, replace
- Top variant per channel → double-down (Spark Ads / boost)

## Sanity check

When you're about to paste a URL, run this mental check:
- ✅ `utm_campaign=mothers_day_2026` (always — same string)
- ✅ `utm_source=` matches the platform you're on (tiktok, pinterest, etc.)
- ✅ `utm_medium=` is one of: `organic` | `paid` | `cpc` | `newsletter`
- ✅ `utm_content=` is specific enough to identify the exact creative
- ✅ Destination path matches the campaign's intended landing page
